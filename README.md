# 运营总监工作台

这是第一版“大框架”，重点是运营总监视角的信息架构和页面骨架。

## 当前包含

- 工作台首页
- 工作台首页已合并数据总览：业绩数据、各项目业绩数据、工作任务、待办事项
- 内容运营
- 内容创作：热点追踪、社群内容、产品知识库、公司知识库
- 数据中心：核心日报、周报、月报、效果优化建议
- 广告计划 / 执行
- 风控监测
- 权限管理

## 当前边界

- 不连接真实后台。
- 不自动发群。
- 不自动置顶。
- 不自动铺设广告。
- 广告计划默认只做 DryRun 和状态展示。

## 打开方式

直接打开 `index.html`，或运行 `启动运营总监工作台.ps1`。

## 后续接入建议

1. 接入已有广告数据面板作为“经营总览 / 日报复盘”的数据源。
2. 接入基金社群 `daily_content.json` 作为“内容运营”的数据源。
3. 接入广告计划执行器，但先保留 DryRun 和人工确认。
4. 把异常、结果、任务状态写回 `data/work-center-state.json` 或本地服务。
